Projects

Exam Platform

An innovative MCQ exam system that supports dynamic question formats, including text, code snippets, and multimedia content. The platform integrates compilers for Python, Java, and SQL, allowing users to execute code seamlessly during the exam. I implemented a partial marking system for questions with multiple correct answers and tracked user activity meticulously, capturing metrics such as time spent on each question and overall exam duration. The result is a detailed post-exam analysis featuring insightful progress graphs. To enhance user experience, I created an intuitive admin panel for dynamic exam creation, enabling both manual and bulk uploads of questions. Additionally, I optimized media storage using GCP, significantly reducing the database load, and introduced a feature that allows users to pause and resume their exams within a specified timeframe, making the platform not only robust but also flexible to user needs.

Collaborative Learning & Discussion Platform

A versatile platform featuring three distinct sections: public discussions, peer group collaborations, and instructor-led faculty rooms. The public discussion area facilitates a Q&A environment where questions are addressed solely by admins or authorized personnel, allowing for rich interactions supported by media attachments. In the peer group rooms, students can collaborate effectively, discussing assignments and sharing progress within their designated groups. The faculty rooms provide a dedicated space for both internal and external instructors to communicate, share study plans, and distribute educational materials seamlessly. To ensure secure communication and robust access control, the platform is hosted on GCP with SSL encryption, creating a safe and reliable environment for all users. Additionally, the platform incorporates user-friendly navigation and intuitive design, enhancing the overall user experience and engagement.
